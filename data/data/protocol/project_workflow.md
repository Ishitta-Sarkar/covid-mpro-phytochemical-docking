# Project Workflow

## Title

Virtual Screening of Plant-Derived Compounds Against SARS-CoV-2 Main Protease

## Aim

To evaluate selected plant-derived bioactive compounds against SARS-CoV-2 main protease using a computational molecular docking workflow.

## Objectives

- Identify SARS-CoV-2 main protease as a therapeutic target.
- Shortlist phytochemicals reported in antiviral and biomedical literature.
- Retrieve protein and ligand structures from public databases.
- Prepare protein and ligand files for molecular docking.
- Perform docking using AutoDock Vina.
- Rank ligands based on predicted binding affinity.
- Summarize the computational findings.

## Methodology Summary

1. Literature review and target identification
2. Protein structure retrieval from RCSB PDB
3. Ligand retrieval from PubChem
4. Protein and ligand preparation using AutoDockTools/Open Babel
5. Docking using AutoDock Vina
6. Docking score comparison
7. Interaction analysis and reporting

## Target Protein

- Protein: SARS-CoV-2 Main Protease
- PDB ID: 6LU7
- Relevance: Important viral enzyme involved in polyprotein processing and viral replication

## Selected Ligands

- Quercetin
- Curcumin
- Kaempferol
- Andrographolide
- Berberine

## Note

This repository presents an educational and computational workflow. Docking results are preliminary and require experimental validation.
